# File Merge Utility

```
usage: debug [<flags>]
Flags:
  -h, --help              Show context-sensitive help (also try --help-long and
                          --help-man).
  -w, --wd="./"           Working Directory
  -i, --inc=".*"          Included file(s)
  -e, --exc="^\\.{1,2}$"  Excluded file(s)
  -o, --out=OUT           Output file name
  -a, --append            Append output result
```
